# 개발 준비 가이드

## 1. 개발 전 준비물

| 준비물 | 필수 여부 | 설명 |
|---|---|---|
| Python 3.11+ | 필수 | AI Server 실행 |
| Java 17 | 선택 | Spring Boot 실행 |
| Maven | 선택 | Spring Boot 빌드 |
| 인터넷 연결 | 선택 | 공개 문서 수집 시 필요 |
| LLM API Key | 선택 | 템플릿 응답 대신 LLM 응답을 쓸 때 필요 |
| 사내 문서 | 선택 | 실제 업무형으로 확장할 때 필요 |

---

## 2. 추천 개발 순서

```text
1. Python AI Server 단독 실행
2. AI4I 모델 학습
3. 샘플 RAG 문서 색인 생성
4. /agent/run API 테스트
5. 공식 문서 수집 스크립트 실행
6. Spring Boot 저장/조회 연동
7. 골든 데이터셋 평가 추가
8. 실제 사내 문서 투입
```

---

## 3. 제조업 도메인 지식이 없어도 기억할 핵심

```text
AI4I는 숫자 데이터로 위험을 예측한다.
RAG 문서는 그 위험에 대해 무엇을 점검해야 하는지 근거를 준다.
Safety Ops는 정비 전 안전 절차를 보강한다.
Report Agent는 답변을 보고서 형식으로 정리한다.
Evaluator는 LLM 답변이 도구 결과와 문서 근거를 왜곡하지 않았는지 검사한다.
```
